👽 UFO Sighting Classifier Web App

This is a Flask-based web application that serves a pre-trained scikit-learn machine learning model to classify UFO sighting data. Users can input various features (e.g., duration, shape) through a simple web form to get a real-time prediction from the model.

 🔗 Live Demo
Deployed App: https://ufo-classifier-app.onrender.com/predict



 🛠️ Technology Stack

Backend Framework: Python (Flask)
Machine Learning: Scikit-learn (Model Training/Prediction)
Data Handling: Pandas, NumPy
Deployment Server: Gunicorn (Production WSGI)
Cloud Hosting: Render (PaaS)
